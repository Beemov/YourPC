## Использование

### Локальный запуск Frontend части

1. Установка зависимостей

```bash
npm install
```

2. Выполните скрипт

```bash
npm start
```

### Локальный запуск Backend части

1. Установка зависимостей

```bash
npm install
```

2. Выполните скрипт

```bash
npm run dev
```
